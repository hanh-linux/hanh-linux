## Hanh Linux
### What is this?
An independent, open-source and source-based Linux distribution. Design for x86_64, build by using [mussel](https://github.com/firasuke/mussel) and use less GNU components 
### Goals
- Fast and lightweight
- Simple
- Flexible
- Non-`systemd`
